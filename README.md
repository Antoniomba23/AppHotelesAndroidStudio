# AppHotel

Aplicación Android (Java)
## Descripción
- Proyecto Android basado en Gradle.
- Estructura estándar con módulo `app`.
- Pensado para ser abierto en Android Studio.

## Tecnologías
- Android SDK, Gradle.
- Kotlin.

## Requisitos
- Android Studio (versión reciente recomendada).
- JDK compatible con la versión de Gradle indicada en el proyecto.
- SDK de Android instalado y configurado.

## Configuración rápida
- Abrir el proyecto en Android Studio: `File > Open...` y seleccionar la carpeta del proyecto.
- Sincronizar Gradle cuando lo solicite.

## Compilación y ejecución
- Desde Android Studio: `Run > Run 'app'` o el botón de ejecutar.

## Estructura del proyecto (resumen)
- `app/src/main/java` código fuente Java.
- `app/src/main/res` recursos de la app (layouts, drawables, strings).
- `gradle/` configuración del wrapper de Gradle.
- `docs/` documentación del proyecto.
